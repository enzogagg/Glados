# Bibliothèque Standard CLaD

Cette section documente l'ensemble des fonctionnalités intégrées au langage CLaD. Ces outils sont disponibles globalement dans votre environnement d'exécution sans nécessiter d'importation explicite.

## Catégories

### 1. [Entrées / Sorties](./StandardIO.md)
Interactions de base avec l'utilisateur et le terminal :
- `print` : Affichage.
- `input` : Saisie.

### 2. [Gestion de Fichiers](./FileIO.md)
Manipulation persistante des données :
- `open`, `read`, `write`, `close`.

### 3. [Structures de Données](./DataStructures.md)
Manipulation de collections complexes :
- **Listes** (`cons`, `head`, `tail`...)
- **Tableaux** (`[...]`, accès indexé)
- **Tuples** (`{...}`)
- **Dictionnaires** (Maps)
- **Structures** (Objets à champs nommés)
