# CLaD Standard Library

This section documents all functionalities integrated into the CLaD language. These tools are available globally in your execution environment without requiring explicit imports.

## Categories

### 1. [Input / Output](./StandardIO.md)
Basic interactions with the user and terminal:
- `print`: Display.
- `input`: Input.

### 2. [File Management](./FileIO.md)
Persistent data manipulation:
- `open`, `read`, `write`, `close`.

### 3. [Data Structures](./DataStructures.md)
Complex collection manipulation:
- **Lists** (`cons`, `head`, `tail`...)
- **Arrays** (`[...]`, indexed access)
- **Tuples** (`{...}`)
- **Maps** (Key-Value)
- **Structs** (Named fields data)
